11:00 - Visualização de dados - Princípios de percepção visual

O termo Visualização de dados se refere a apresentação de informações e dados em formato gráfico. Usando elementos visuais como diagramas, gráficos e mapas, a visualização é uma forma acessível de ver e entender fenômenos como exceções, tendências e padrões nos dados. Nesta apresentação pretendo apresentar uma introdução a Visualização de Dados e alguns princípios de percepção visual que nos ajudam a entender o que torna uma visualização eficaz.
